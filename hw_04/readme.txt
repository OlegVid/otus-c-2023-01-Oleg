ПриложениПринимая в качестве аргумента наименование города, приложение выводит на экран прогноз погоды на текущий день, город (на англ языке),
 текстовое описание погоды, направление и скорость ветра, температура.

Для успешной сборки должны быть установлены следующие библиотеки:

curl (libcurl)  
				Команды установки:
				 sudo apt-get install libcurl-dev
				 sudo apt-get install libcurl4-nss-dev
				 sudo apt install libcurl4-gnutls-dev

json-c      
				Команды установки:    
				sudo apt install libjson-c-dev

pkg-config    
				Команды установки:
				 sudo apt install pkg-config
