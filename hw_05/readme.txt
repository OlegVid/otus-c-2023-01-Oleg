Проект создания библиотеки для логгирования событий внутри программы
События записываются подряд в задаваемый пользователем при старте логгирования файл, в формате XML.

В каталоге ll_log - файлы библиотеки логирования

littlelog.h - заголовочный файл к библиотеки в том числе содержит интерфейсные 
function-like макроссф для работы с ней.

в каталоге ll_log находится Make файл для сборки и архивации этой библиотеки.
Для ее пересборки нужно вызвать терминал в этом каталоге и набрать make
соберется файл libllog.a

в корневом каталоге находится файл main.c - который содержит исходники программы для
демонстрации работы библиотеки.

В корневом каталоге проекта - также находится  Make файл через который собирается проекта
при сборке статически линкуется библиотека из каталога ll_log

в программе демонстраторе - запускаются три потока которые начинают с периодом 10 мс 
писать сообщения в лог.

по окончании работы потоков, файл лога закрыватся и программа завершает работую.
ПО умлочанию программа записывает лог файл в файл test_log.xml в корневом каталоге,
 но в первом параметре вызова можно задать альтернативное имя файла.
 например ./hw_05 ./another_log_name.xml
